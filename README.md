This repository contains the code used to analyze bacteriophage resisitance evolution of the honey bee pathogen, P. larvae. The paper associated with this project is available here:

Bacteriophage resistance evolution in a honey bee pathogen
Emma Spencer, Tracey Peters, Yva Eline, Lauren Saucedo, Kevin Linzan, Keera Paull, Craig Miller, James Van Leuven
doi: https://doi.org/10.1101/2024.07.09.602782
